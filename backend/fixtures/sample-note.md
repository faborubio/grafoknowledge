# Grafos de conocimiento

Los grafos de conocimiento representan entidades y sus relaciones de forma explícita.
Neo4j es la base de datos de grafos que estoy usando en el proyecto GrafoKnowledge.

## Ideas

- GraphRAG combina recuperación sobre grafos con generación por LLM. Es prerequisito
  entender embeddings antes de meterse con GraphRAG.
- El libro "Designing Data-Intensive Applications" de Martin Kleppmann trata sobre
  sistemas de datos distribuidos y me sirvió para pensar el modelo de datos.
